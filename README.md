# Estudos de Java

Este repositório foi criado em sala de aula com o objetivo de ensinar os conceitos fundamentais de programação em Java, pertinentes à disciplina de Programação Web I.

O projeto utiliza **Java 17** e ainda está em construção, sendo atualizado conforme o andamento das aulas.

Ele reúne exemplos práticos desenvolvidos em sala e a correção das atividades propostas aos alunos.

---

## Estrutura do Projeto

O projeto está organizado da seguinte forma:

### 📦 exemplos

Contém os códigos desenvolvidos em sala de aula. Esses arquivos servem como material de apoio para estudo e revisão.

### 📦 atv1

Contém a correção da primeira atividade proposta em sala.

---

## Como executar?

1. Clone o repositório:
```sh
git clone https://github.com/almeidaluciana/pwi-java.git
```
2. Abra o projeto na sua IDE (recomendado: IntelliJ).
3. Execute a classe desejada dentro dos pacotes `exemplos` ou `atv1`.
